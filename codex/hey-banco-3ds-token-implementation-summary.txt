Hey Banco / Banregio — 3D Secure con token guardado (Camino B)
================================================================
Fecha: 2026-06-09

QUÉ SE IMPLEMENTÓ
-----------------
Flujo 3DS para cobros con tarjeta tokenizada (BNRG_TOKEN) sin enviar PAN/exp/CVV.
Detrás de HEYBANCO_3DS_ENABLED=false por defecto. No modifica EfevooPay, Stripe,
PayPal ni Odessa. No llama ChargeHeyBancoTokenAction tras callback aprobado.

FLUJO FINAL
-----------
1. Checkout laboratorio con hey_banco:{id} y HEYBANCO_3DS_ENABLED=true
2. OrderAction → InitiateHeyBanco3dsLaboratoryCheckoutAction
3. Crea PaymentAttempt (pending_3ds), PaymentTransaction (token_3ds_charge), Payment3dsSession
4. POST a HEYBANCO_3DS_URL con BNRG_TOKEN (sin datos de tarjeta)
5. Redirect interno → HeyBanco3dsRedirect → URL Banregio
6. Usuario completa challenge
7. Banregio POST a BNRG_URL_RESPUESTA (callback)
8. Valida BNRG_HASH si HEYBANCO_3DS_SECURE_API=true
9. Si BNRG_CODIGO_PROC=A: crea Transaction, fulfillment laboratorio (sin chargeToken)
10. Usuario ve HeyBanco3dsResult / redirección al pedido

FIRMA REQUEST
-------------
HMAC-SHA256 + Base64 con HEYBANCO_3DS_SECRET_KEY.
Campos concatenados en orden de config('heybanco.3ds_request_sign_fields').
TODO: confirmar orden exacto con Manual Colecto 3DS V1.2 (no disponible en repo).

VALIDACIÓN RESPONSE
---------------------
Mismo algoritmo con config('heybanco.3ds_response_sign_fields').
Valores URL-decoded antes de concatenar.

EVITAR DOBLE CARGO
------------------
- 3DS aprobado NO invoca ChargeHeyBancoTokenAction / chargeToken ADQ.
- El cargo lo realiza el motor Colecto durante el flujo 3DS.

EVITAR DOBLE FULFILLMENT
------------------------
- Callback con lockForUpdate en payment_3ds_sessions.
- Si sesión ya approved → already_processed=true, no re-ejecuta fulfillment.
- Transaction idempotente por payment_3ds_session_id en details.

INTEGRACIÓN CHECKOUT
--------------------
OrderAction.chargeAndCreateTransaction detecta hey_banco + 3ds_enabled.
Lanza HeyBanco3dsRedirectRequiredException → LaboratoryPurchaseController
redirige a payments.hey-banco.3ds.redirect.
